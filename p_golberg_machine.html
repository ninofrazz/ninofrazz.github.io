SCI-FI RUBE GOLDBERG MACHINE

Sc-fi themed Goldberg machine made on Unity.



Date: 2019
Purpose: Game Development Pipeline for BSc
Subjects: Programming
Programs: Unity, Visual Studio (C#)

The Concept

To ensure the functionality of this Goldberg machine, I utilized triggers as the primary mechanism. These triggers are designed to activate scripts when the ball comes into contact with specific elements, initiating a chain reaction of events throughout the machine.

Devices

Below are the interactions that have been implemented in the Goldberg machine:

  • Ball controller
  • Buttons
  • Bucket
  • Gravity Changer
  • Pushing Piston
  • Seesaw
  • Spinning Sticks
  • Pendulum
  • Domino set
  • Catapult
  • Vent
  • Teleport
  • Dash Pad
  • Chained Boxes
  • Spinner
  • Spring
  • Ball Spawner
  • Counter and Exploder
  • Finish Banner
